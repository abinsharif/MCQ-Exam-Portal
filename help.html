Generating a Base64 Exam String

Use an AI (Grok, Claude, Gemini, ChatGPT) with this prompt to get a Base64 string:

Create multiple choice questions in valid JSON format. Each question must have exactly 4 options, one correct answer, and a brief explanation. Subject: Mathematics Chapter: Logarithm Number: 30 Difficulty level: medium Language: English Make questions SSC level standard. Required JSON structure: [ { "question": "Question text here", "options": ["a) Option A", "b) Option B", "c) Option C", "d) Option D"], "answer": "Correct option text (must match one option exactly)", "solution": "Brief explanation" } ] After creating the JSON, encode it using Python: import base64, json data = [PASTE_JSON_HERE] encoded = base64.b64encode(json.dumps(data, ensure_ascii=False).encode('utf-8')).decode() print(encoded) Give me ONLY the final Base64 string.
Critical for Bangla/Unicode: Always use ensure_ascii=False and encode('utf-8') in the Python code. This preserves Bangla text and math symbols (²³×÷π).

Replace: Subject, Chapter, Number (how many questions), Difficulty (easy/medium/hard), Language (Bangla/English/Mixed)

JSON Format Reference

Each question must follow this structure exactly:

[ { "question": "What is 2³ × 2⁵?", "options": ["a) 2⁸", "b) 2⁵", "c) 2⁷", "d) 2³"], "answer": "a) 2⁸", "solution": "When multiplying same base, add exponents: 3+5=8 → 2⁸" } ]
  • question — The question text (supports Bangla, Unicode, math symbols)
  • options — Array of exactly 4 options
  • answer — Must match one option exactly (copy-paste!)
  • solution — Optional explanation (shown in results)

Keyboard Shortcuts

1 2 3 4Select option A/B/C/D on first question of page
SpaceGo to next page (when pages enabled)
EnterSubmit exam
PPause / Resume (when pause button is enabled in Settings)
Ctrl+EnterStart exam from Base64 input

Shortcuts can be disabled in Settings.

Settings Guide

  • Your Name — Displayed on result screen
  • Questions Per Page — Show 1, 2, 5, 10, 15, 20, 30, or All questions at once
  • Practice Mode — Exams won't be saved to history
  • Show Pause Button — Hidden by default, enable to show pause/resume
  • Font Size — Small / Medium / Large / X-Large
  • Sound Effects — Click sounds, last-minute warning, result fanfare
  • Keyboard Shortcuts — Enable/disable all keyboard shortcuts
  • Clear All Data — Wipes name, settings, history, and autosave
